# Decision-tree-in-SAS-Enterprise-Miner
Implementing Decision Tree on Iris Dataset

Following are the steps to create the example
1. Create Project
2. Create Diagram
3. Drag File Import node from Sample tab of the SAS Enterprise Miner toolbar
4. Import Dataset and Select Target variable
5. Drag Data Partition node from Sample tab of the SAS Enterprise Miner toolbar
6. Drag Decision Tree node from Model tab of the SAS Enterprise Miner toolbar
7. Run the Cluster node and view the results
